# Weather-App
Project Name: Weather App (Professional Version)


The Weather App is a Python-based GUI application that allows users to check real-time weather information for any city. Users can type a city name, click “Get Weather,” and instantly see details such as temperature, “feels like” temperature, humidity, and weather conditions. The app also displays weather icons, updates the background color based on weather type. It is beginner-friendly, visually interactive, and demonstrates the use of APIs, error handling, and GUI development with Tkinter


Features:

  GUI input box for city

2.   Displays temperature, humidity, weather condition

3.   Shows feels like temp, last updated time

4.   Weather icons and background changes

5.   Error handling


Installation Instructions:
git clone https://github.com/yourusername/weather-app.git
cd weather-app
pip install -r requirements.txt
python weather_app.py


Skills Learned: Python, Tkinter, API usage, Error handling, GUI design
